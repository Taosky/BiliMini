## BiliMini
Bilibili辅助扩展，用于提示并快速查看更新。

### 安装
**Firefox**: https://addons.mozilla.org/zh-CN/firefox/addon/bilimini/

**Chrome**: https://github.com/Taosky/BiliMini//releases/latest


### 功能
- 角标数字提醒
- 展开更新列表
